## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript.


RESTful api project, perform a CRUD on an array of objects containing cars.

Path /cars

Methods
- HTTP request: GET http://localhost:3000/cars

- HTTP request: POST http://localhost:3000/cars/
- Body
{
  "brand": string,
  "model": "string",
}

- HTTP request: PATCH http://localhost:3000/cars/{id}
- Body
{
  "brand"?: string,
  "model"?: "string",
  "id"?: "uuid()"
}
- HTTP request: DELETE http://localhost:3000/cars/{id}

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
